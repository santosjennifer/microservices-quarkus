## Microserviços com Quarkus

Projeto ifood, contendo serviços de Cadastro, Pedido, Marketplace e Entregador.

### Tecnologia:

- Quarkus 3.11.1
- Flyway
- PostgreSQL
- MongoDB
- ElasticSearch
- Apache Kafka
- ActiveMQ Artemis
- Jaeger
- Logstash
- Kibana
- Keycloak
- Prometheus
- Grafana

**Observação:** Acesse o readme de cada serviço para maiores informacões.
